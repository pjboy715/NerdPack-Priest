# NerdPack-Priest
该项目是基于Nerdpack的一个牧师自动战斗策略。

建议：
使用神牧，根据合理的设置以及脚本的迅速反应特性，可以轻松的碾压一众治疗。
戒律牧在不能做有效的预判前，真垃圾。戒律牧擦屁股也垃圾。戒律牧队伍模式挂挂救赎还行吧。

目前神牧做的还行，戒律牧吧，能奶死人。暗牧压根儿没做。
神牧部分，建议使用 神牧-团本（测试环境是30人团）。支持gui设置各个技能的施放条件

1.[神牧] 治疗祷言，如何获取40码内距离最近的4个队友？
